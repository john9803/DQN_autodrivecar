Auto_Drive_Car_Project
Park Jin Young
-------------------------------------------------------------------------------------
< 모델의 기반 >
OpenAI gym
Car_Racing-v0의 env를 기본적으로 사용하는 프로젝트임.
-------------------------------------------------------------------------------------
< 구조 >
기본적인 구성은 train_mine.py와  test_mine.py파일로 구성되어 있으며 
각 case별로 학습 그래프와 에피소드에 따른 모델이 저장되어 있음.
-------------------------------------------------------------------------------------
< 필요한 패키지 환경구축 >
필요한 패키지들은 requirements.txt 파일에 들어있으며,
설치 할 때는 'pip intsall -r requirements.txt' 명령어를 
이용해 가상환경에 패키지를 설치
-------------------------------------------------------------------------------------
<tensorflow 버전>
GPU보다 CPU가 학습이 더빠른 점도 있는것 같아 AVX명령어를 사용해 빠르게
학습을 진행하는 tensorflow-cpu 버전을 따로 설치해서 진행하였음.

참고자료 
https://ryudaewan.wordpress.com/2018/10/28/10349/
https://github.com/fo40225/tensorflow-windows-wheel

--------------------------------------------------------------------------------------
< 실행 방법 >
test_mine.py 에서 agent.load메소드를 통해 학습된 모델을 불러올 수 있으며,
경로설정은 현재 test_mine.py가 포함되어 있는 auto_drive_car_project 디렉토리
이하의 디렉토리를 입력.
Ex) agent.load("./case7/save_model/auto_drive_train_Episode_500.h5")
와 같이 경로를 지정하여 사용한다.
